Validator
$('#form1').validatable();
$('#form1').validatable(
	{ global options
		bind: {
			change: false, ignore onchange event
			blur: true
		},
		className: {
			valid: 'has-success', class applied to a valid form-group
			invalid: 'has-error' class applied to an invalid form-group
		}
	},
	{ element options (keys match form, group or element id attribute)
		form1: {
			observe: function(type, form, event){ form event observer
				if ('form.submit' == type) {
					alert("Form #"+form.id+" cannot be submitted.");
					return false; returning "false" stops event propagation
				}
				return true;
			}
		},
		group1: {
			debug: true enables console.debug logging for this group and its elements
		},
		text1: {
			required: false, field is optional
			rules: {
				regex: /^07[0-9]{8,9}$/ field must match the regular expression to be valid
			}
		},
		select1: {
			validate: function(field){ replaces build-in validation
				return field.val() == 'option1'; can return true (valid), false (invalid) or undefined (neutral)
			}
		},
		datepicker1: {
			datepicker: { options passed to jQuery-UI datepicker
				altFormat: 'yy-mm-dd',
				dateFormat: 'dd/mm/yy'
			}
		},
		group4: { group must contain at least one, but no more than 3 valid fields
			rules: {
				min: 1,
				max: 3
			}
		}
	}
);
Options

Global options are merged into form, group and field options.

Options can be set globally (as a property of the first parameter of validatable) or attached to a group/field.

option allowed elements
debug enables console.debug messages for element and its children
required group must contain at least one valid field
field must be valid
bind.change enables onchange event propagation, including group validation
bind.blur enables onblur event propagation, including group validation
bind.keydown enables onkeydown event propagation
bind.submit enables onsubmit event propagation, including form validation
className.valid class applied to form-group div on successful group validation (default: has-success)
className.invalid class applied to form-group div on failed group validation (default: has-error)
observe attaches observer to element
validate attaches validator to element
rules.val required field value
rules.regex required field regular expression value
rules.min required minimal number of valid fields within group
rules.min required maximum number of valid fields within group
datepicker.* options passed to jQuery-UI Datepicker

Observers

The event propagation is stopped if the observer returns strict false.

An observer must be attached via observe option and accept three parameters:

  • type - event type (string)
  • obj - either Form, Group or Field object
  • event - Event object (for default browser event)
$('#form1').validatable({}, {
	text1: {
		observe: function(type, field, event){
		}
	}
});
Validators

Custom validators take precedence over build-in validation and any custom rules.

A validator can be attached via validate option and accept one parameter:

  • obj - either Form, Group or Field object

A validator can return one of the following values:

  • strict true - the element is valid
  • strict false - the element is invalid
  • strict undefined - the element is either empty but optional, has not been validated or its state is unknown
$('#form1').validatable({}, {
	text1: {
		validate: function(field){
			var value = field.val();
			return (value >= 1 && value <= 3);
		}
	}
});
Rules

Custom rules take precedence over build-in validation.

Predefined validation rules can be attached to a group or field via rules option.

If there is more than one rule attached to an element, all of them must be fulfilled for the element to be considered valid.

element rule name value type description
groupminintminimum number of valid fields (for instance selected checkboxes)
groupmaxintmaximum number of valid fields (for instance selected checkboxes)
fieldvalintrequired field value
fieldregexRegexfield value must match the provided regular expression
 
$('#form1').validatable({}, {
	group1: {
		rules: {
			min: 1,
			max: 3
		}
	}
});
$('#form1').validatable({}, {
	text1: {
		rules: {
			regex: /^0[0-9]{9,10}$/
		}
	}
});
